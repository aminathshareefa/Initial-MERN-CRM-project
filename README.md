# MERN CRM Application

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Authentication (Register/Login)
- Customer CRUD Operations
- JWT-based Authorization
- Responsive UI

## How to Run
1. Backend:
   - cd backend
   - npm install
   - npx nodemon server.js

2. Frontend:
   - cd frontend
   - npm install
   - npm start
